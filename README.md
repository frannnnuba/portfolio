# portfolio
Hello! This is my backend portfolio built with Spring Boot (Java). 

##Projects:

### Store-api
A backend application that simulates the core logic of an e-commerce system.

**Features:**
- Product catalog (public access)
- User authentication & authorization (Spring Security + JWT)
- Role-based access (ADMIN / USER)
- Shopping cart persistence
- Order creation (checkout flow)
- RESTful API design

**Tech stack:**
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- MySQL

This project focuses on backend logic and architecture. 
(No frontend included for now, but planned in the future.)
